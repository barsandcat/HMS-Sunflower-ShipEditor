## Coding standards and style
- Follow the project's `.editorconfig` where present for spacing and indentation.

## Naming
- Type prefixes: none (p, b, etc) - compiler will help you.
- Type names: Prefix+PascalCase
- All class members: PascalCase
- Function names: PascalCase
- Constants: SCREAMING_SNAKE_CASE
- Local variables, function parameters: snake_case
